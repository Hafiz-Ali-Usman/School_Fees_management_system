# School_Fees_management_system
**School Fees Management System** is a production-ready desktop application built with **Electron**, **Node.js**, and **SheetJS** that enables schools to efficiently manage student fee records, track payment installments, and generate detailed financial reports. The system stores all data in separate Excel files (one per class) for easy sharing and offline access, providing both a command-line interface (CLI) and graphical user interface (GUI) for managing multiple student entries, recording payments, and viewing comprehensive payment histories. Features include real-time fee calculations, installment tracking with dates and notes, dashboard statistics, and the ability to handle multiple payment entries per student with automatic balance computations.
